# Caffeine Check

A personal caffeine tracker that helps you stay aware of your daily intake — without the anxiety.

**[→ Live demo](https://YOUR-USERNAME.github.io/caffeine-check)**

![Caffeine Check screenshot](screenshot.png)

## What it does

- Calculates a personalised daily reference based on your weight, age, and caffeine sensitivity
- Logs drinks across branded options (Starbucks, Red Bull, etc.), custom coffee, tea, and soft drinks
- Shows a live arc gauge of your intake vs. your limit
- Previews how your next drink would affect your total before you log it
- Tracks active caffeine in your body using a half-life decay model
- Warns you if caffeine will still be elevated by your bedtime

## Built with

- React 18 (via CDN, no build step)
- Vanilla SVG for the gauge
- Zero dependencies beyond React

## Run locally

Just open `index.html` in a browser. No server or build step needed.
